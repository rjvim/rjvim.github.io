## Configure docusaurus

```typescript title="docusaurus.config.ts"
{
  organizationName: 'orgName-or-Username', // Put your GitHub org/user name.
  projectName: 'projectName', // Put your repo name.
}
```

## Deploy to Github Pages

The first deployment to Github pages is manual, but we will automate it in the next steps.

```bash
npm run docs:deploy
```

After running this command you should see a branch in your Github like following:

![gh-pages branch](./gh-pages-branch.png)

## Configure Domain

Depending on to which endpoint you want to deploy you have to configure baseUrl accordingly and do DNS mapping for your preferred domain. If you are going to use `*.github.io` you won't have to configure DNS. But you would need to configure `baseUrl`.


```typescript title="docusaurus.config.ts"
{
  baseUrl: "???"
}
```
