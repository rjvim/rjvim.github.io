# GithubCodeBlock Component

The `GithubCodeBlock` component allows you to embed and display code from GitHub repositories directly in your application. It supports line extraction, syntax highlighting, and line highlighting features.

## Basic Usage

```tsx
<GithubCodeBlock url="https://github.com/rjvim/react-component-library-starter/blob/main/package.json" />
```

## Props

The component accepts the following props:

| Prop                 | Type      | Default     | Description                                                    |
| -------------------- | --------- | ----------- | -------------------------------------------------------------- |
| `url`                | `string`  | (required)  | GitHub URL to the file you want to display                     |
| `extractLines`       | `boolean` | `false`     | Whether to extract specific lines from the file                |
| `highlightLines`     | `string`  | `undefined` | Lines to highlight in the format `"{1,3-4}"`                   |
| `useLocForHighlight` | `boolean` | `true`      | Whether to use the location hash from the URL for highlighting |

## Examples

### Display a Complete File

```tsx
<GithubCodeBlock url="https://github.com/rjvim/react-component-library-starter/blob/main/package.json" />
```

### Extract Specific Lines

You can extract specific lines from a file by adding a line reference to the URL and setting `extractLines` to `true`:

```tsx
<GithubCodeBlock
  url="https://github.com/rjvim/react-component-library-starter/blob/main/package.json#L8-L16"
  extractLines={true}
/>
```

### Highlight Specific Lines

You can highlight specific lines using the `highlightLines` prop:

```tsx
<GithubCodeBlock
  url="https://github.com/rjvim/react-component-library-starter/blob/main/package.json"
  highlightLines="{2,8-16}"
/>
```

### Extract and Highlight Lines

When `extractLines` is `true`, highlighting is disabled to ensure clean display of extracted code snippets:

```tsx
<GithubCodeBlock
  url="https://github.com/rjvim/react-component-library-starter/blob/main/package.json#L8-L16"
  extractLines={true}
  highlightLines="{2}" // This will be ignored when extractLines is true
/>
```

### Use URL Location for Highlighting

By default, the component will use the line references in the URL hash for highlighting when `useLocForHighlight` is `true`:

```tsx
<GithubCodeBlock
  url="https://github.com/rjvim/react-component-library-starter/blob/main/package.json#L8-L16"
  useLocForHighlight={true} // Default behavior
/>
```

To disable this behavior and only use the `highlightLines` prop:

```tsx
<GithubCodeBlock
  url="https://github.com/rjvim/react-component-library-starter/blob/main/package.json#L8-L16"
  useLocForHighlight={false}
  highlightLines="{2,5}"
/>
```

## Highlight Format

The `highlightLines` prop accepts a string in the format `"{lineNumber,lineRange}"` where:

- Single lines are specified as numbers: `"{1,3,5}"`
- Ranges are specified with a hyphen: `"{1-3,5-7}"`
- You can combine both: `"{1,3-5,7,9-11}"`

## Example with Multiple Features

```tsx
<GithubCodeBlock
  url="https://github.com/rjvim/react-component-library-starter/blob/main/package.json#L8-L16"
  extractLines={false}
  highlightLines="{2,8-16}"
/>
```

This will display the entire file from the GitHub URL but highlight lines 2 and 8-16.
